This page provides a complete guide to the analytical code and computational resources associated with the MetaCheeseDB manuscript. All resources are publicly available to ensure full reproducibility of the reported findings.

---

## Bioinformatics pipelines

All pipelines for raw read preprocessing, taxonomic profiling, genome assembly, binning, quality assessment, and functional annotation are available at [SegataLab/MASTER-WP5-pipelines](https://github.com/SegataLab/MASTER-WP5-pipelines).

Please, refer to the following modules:

- `02-Preprocessing` for raw read quality control and trimming;
- `03-Assemblyfree_analysis` for taxonomic profiling;
- `04-Strainlevel_analysis` for strain-level phylogenomics
- `05-Assembly_pipeline` for assembly, indexing and mapping, and binning;

---

## Statistical and machine learning analysis scripts

All R scripts for the statistical analyses, machine learning, and explainable artificial intelligence analyses described in the manuscript are available in the MetaCheeseDB GitHub [Code and reproducibility page](https://magliulo.github.io/metacheesedb/jekyll/2026-03-19-code.html).

| Script                                                      | Analysis                                                                                 |
|-------------------------------------------------------------|------------------------------------------------------------------------------------------|
| `01_setup_and_data_loading.R`                               | Environment setup, library loading, data import                                          |
| `02_basic_statistics_and_core_microbiome.R`                 | Descriptive statistics and core microbiome analysis                                      |
| `03_alpha_diversity.R`                                      | Alpha diversity metrics and statistical comparisons                                      |
| `04_beta_diversity_ordination.R`                            | Beta diversity, PERMANOVA, t-SNE, UMAP                                                   |
| `05_composition_visualisation_and_differential_abundance.R` | Heatmaps, pairwise comparisons (Wilcoxon, Kruskal-Wallis, Benjamini-Hochberg FDR), LEfSe |
| `06_supervised_machine_learning.R`                          | Random Forest classifier, One-vs-Rest AUC-ROC, bootstrapped confidence intervals         |
| `07_SHAP_explainability.R`                                  | SHAP values and beeswarm visualisations                                                  |
| `08_genome_quality_and_taxonomy.R`                          | CheckM2, GTDB-Tk, dREP                                                                   |
| `09_functional_analysis.R`                                  | Functional gene annotation across all categories                                         |

## Reproducibility notes

- Input data (MetaPhlAn taxonomic profiles and curated metadata) required to run the R scripts are available via the [MetaCheeseDB browsing interface](https://magliulo.github.io/raffaelemagliulo/files/MetaCheeseDB.html) and the Metadata page of this website.
- Scripts should be run sequentially, as downstream scripts depend on objects created in earlier ones.
- Update the `setwd()` and file paths in `01_setup_and_data_loading.R` to match your local environment before running.
